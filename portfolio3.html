Building a Simple and Responsive Website Using Bootstrap

And the role of development in design

Bootstrap is a free and open-source front-end framework for HTML, CSS, and Javascript. It allows developers to quickly create mobile-first, responsive web projects by leveraging a library of prebuilt components, grids, forms, buttons, icons, navigation, and other interface elements.

This is a deep-dive into the role of development in the design process, with a focus on responsive design. This project was created in one of my web design class. From this class, I learn foundational web design principles by planning, developing, designing, and testing various web-based media.
